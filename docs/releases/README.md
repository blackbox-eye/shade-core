# Releases

Denne mappe samler den repository-nære del af releasearbejdet.

## Fokus

- Forberedelse af ændringer og release-relateret dokumentation.
- Korte kriterier før merge til `main`.
- Kun lokale og repository-relevante trin.

## Arbejdsramme

- Arbejd lokalt i VS Code med PowerShell som shell.
- Opret branch, commit med signering og åbn pull request.
- `main` er protected, og merge sker kun via pull request.
- Historikken skal være lineær.
- Deploy, VPS og secrets er uden for scope.
